# TotalAIChat
## 功能介绍
- Qt6 +QWebViewEngine+C++
- 集成了8个国内主要的聊天机器人(问心一言、腾讯元宝、通义千问、豆包、Kimi、开工、360、讯飞)
- 一个输入框，同时发送给8个AI机器人
- 需要自己去登录8个AI机器人
- 有网页是禁用了devtool,可以禁止
